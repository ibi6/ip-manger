# 答辩演示步骤（大约 8 分钟）

## 开始前

1. 后端 8000、前端页面都打开  
2. 准备 admin / ChangeMe123!

## 步骤

1. **说明题目**（30 秒）  
   管企业内网 IP，减少 Excel 撞地址、难回收的问题。

2. **登录**  
   用 admin 进系统，提一下还有网管、部门、只读三种角色。

3. **工作台**  
   看地址数量、空闲、冲突数。

4. **子网**  
   点开一个子网，说明创建时会生成地址池；网络地址/网关是预留的。

5. **分配**  
   选一个空闲 IP，填主机名分配。再到地址台账搜一下。

6. **回收**  
   把刚才的地址收回 free。

7. **用户管理**（管理员）  
   简单加一个用户或停用某个账号，证明有用户维护。

8. **冲突记录**  
   可以说：这里登记冲突；扫描按钮是演示用的模拟数据，不是真扫网。

9. **收尾**  
   技术栈：Python FastAPI + React + SQLite；不足：设备/VLAN 还没独立模块、探测是模拟的。

## 老师如果较真

- 问 Flask：直接说本项目用的是 FastAPI。  
- 问 MySQL：演示 SQLite，结构用 SQLAlchemy，可换 PostgreSQL/MySQL。  
- 问设备管理：目前是地址上的字段，后续可拆 devices 表。
