## 第2章 相关技术
每项写“是什么 + 我为什么用”，各半页即可。

- Python / FastAPI：做 REST 接口方便，自带 OpenAPI 文档  
- SQLAlchemy：表结构和 Python 类对应，少手写 SQL  
- SQLite：演示零配置；生产可换 MySQL（schema 见 docs/schema.sql）  
- React + TS：页面状态多，组件化好改  
- JWT：前后端分离下做登录态  
- bcrypt：密码不明文存

| 站点/子网 | CIDR、网关、VLAN号字段 |
